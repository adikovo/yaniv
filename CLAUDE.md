# Yaniv Card Game

A real-time multiplayer card game built with React + Vite (client) and Express + Socket.io (server).

## Structure
- `client/` — React 19, React Router, Socket.io-client, Vite
- `server/` — Express, Socket.io

## Dev Commands
```bash
# Client
cd client && npm run dev

# Server
cd server && npm start
```
